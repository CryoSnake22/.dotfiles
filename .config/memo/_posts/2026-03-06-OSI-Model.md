# OSI Model

## Layer 1 : Physical

The actual electric and radio signals

## Layer 2 : Data Link Layer

Communication between devices on the same network (Only MAC addresses)

## Layer 3 : network

This is where Ip Addresses live

## Layer 4 : Transport

This is where TCP, UDP lives

## Layer 5 : Session

Manages sessions between systems

- Socks proxies get associated with this apparently

## Layer 6 : Presentation

Encryption, compression, serialization formating

## Layer 7 : Application

The full on protocols used by the applications:

- HTTPS
- DNS
- FTP
- HTTP
- SSH
